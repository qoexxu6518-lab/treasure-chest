宝箱系统
一系统概述

宝箱系统是各类资源赏、进度赏与活动赏的统一发放与开启容器。本系统为完全独立的玩法系统，不依赖任何其他活动生命周期；其他系统产出的宝箱会入库到本系统，玩家可随时进入宝箱界面选择开启时机。

  • 宝箱按 5 个品质阶段划分：木制 / 青铜 / 白银 / 黄金 / 钻石
  • 宝箱仓库本身只负责存储、开启与进度计量；宝箱来源交由各产出系统控制
  • 本系统包含宝箱积分循环与独立等级两条赋能线：积分循环产出奖励宝箱、等级决定奖励宝箱品质，详见章节二
1.1 系统入口（主界面集成）
本系统不新增主界面入口样式，仅复用现有主界面右侧的活动按钮区接入入口；按钮位置、外观、出现时机都交由主界面文档统一控制，本文档仅说明点击后的入口逻辑。
⚠ 宝箱仓库入口可同时在多个主界面（例如封地主界面、其他主界面）出现，以保证玩家在不同场景下都能随时点进仓库查看与开启宝箱。多处入口指向同一个宝箱仓库界面，状态（库存、等级、积分、赛季）完全共享。
⚠ 入口外观基于主界面调整后的设计。若主界面尚未完成调整，先在大致位置摆放临时按钮占位（满足跳转逻辑即可），待主界面定稿后替换为正式样式。

一系统概述（原）

攻城掠地是国战地图活动中的阶段结算奖励载体。每日活动划分为 3 个阶段，活动主界面随阶段切换形态：

  • 准备期（04:00–20:00）：展示战场地图 + 玩法简介，供玩家预览局势与了解规则
  • 战斗期（20:00–22:00）：玩家围绕 9 个都城展开攻防；主界面同步展示实时地图 + 国家/个人积分
  • 领奖期（22:00–次日 04:00）：阶段结算后由玩家点击宝箱领取，未领取者 04:00 邮件补发

阶段结束（22:00）时系统按以下两个维度为玩家分配宝箱：

  • 国家总体得分 —— 决定宝箱品质（白 / 绿 / 蓝 / 紫 / 橙，共五档）
  • 玩家个人得分 —— 决定宝箱数量（由数值配置定义，最高三位数）

宝箱领取后进入独立的宝箱界面，玩家可延迟开启，无强制时限。

⚠ 原型中三方国家以"魏 / 蜀 / 吴"作为演示名称，仅用于占位。实际三国名称（含国旗、配色、阵营图腾等）以国战玩法的最终设定为准，本系统仅消费"国家归属"这一抽象概念。

2.3 国家积分计分（地图维度）
⚠ 本节中所有带橙色虚线下划线的数值（如积分、阈值、倍率等）均为演示用占位值，仅用于帮助理解规则；最终数值以数值策划配置表为准。

每日 20:00–22:00 战斗时段，地图共 9 个都城。以玩家本国为视角，9 座都城按归属 / 争夺难度划分为三类，越难拿下的都城分越高：

类型含义包含都城（魏视角举例）积分（演示）
关内本国本土都城，最好守晋阳、洛阳（本国 2 座）1,000 / 座
关外三国交界中立都城，争夺热点徐州、长安、苍梧（3 座三国同为“关外”）3,000 / 座
敌境另两国本土都城，远攻加分南海、建业（吴本土）、永昌、汉中（蜀本土）8,000 / 座
⚠ 分类以玩家的本国为参照：同一座都城对 A 国是关内，对 B 与 C 国则是敌境。例：“南海 / 建业”是吴国的关内，对魏、蜀是敌境；“永昌 / 汉中”是蜀国的关内，对魏、吴是敌境。三国玩家看到的“关内 / 敌境名单”各不相同，但关外都城（徐州 / 长安 / 苍梧）三国一致。

战斗时段内城池可被反复争夺，但不计算占领时长，仅以 22:00 结算时刻各城池的归属为准。国家积分 = ∑ 本国最终占领的城池积分。

2.4 个人积分计分（战斗维度）
⚠ 本节中所有带橙色虚线下划线的系数为演示值，最终数值以数值策划配置表为准。

个人积分由击杀得分与损兵补偿两部分构成，且仅统计 9 座都城范围内产生的杀敌与伤亡，都城以外的战斗不计入：

行为结算方式系数（演示）
击杀敌方部队（都城内）按消灭兵力总价值×1.0
损失己方部队（都城内）按阵亡兵力总价值（补偿）×0.3

即使阵亡损失也可换算为部分积分，鼓励主动出击。

2.5 奖励结算逻辑：国家定品质，个人定数量
⚠ 本节中所有带橙色虚线下划线的档位阈值与数量均为演示值，最终数值以数值策划配置表为准。

三主界面 · 三阶段形态
3.1 总体布局

活动主界面的外框结构（顶部活动标题 / 副标题 + i 按钮、底部活动倒计时）在三个阶段下保持一致；主体内容区会随阶段切换三种不同形态：

  • 顶部：活动标题 "攻城掠地" + 赛季副标题 + 右上角 i 按钮（任意阶段点击均打开详细规则弹窗）
  • 底部：活动倒计时（提示距离下一阶段开始的时间）
  • 中部主体：随阶段呈现 准备期 / 战斗期 / 领奖期 三种形态（详见 3.2 ~ 3.4）
3.2 准备期形态（04:00 – 20:00）

主体显示战场地图+玩法简介，让玩家提前了解战局与规则：

  • 战场地图（缩略）：展示 9 座都城及其当前归属（通过颜色区分魏 / 蜀 / 吴 / 无主），以及图例说明
  • 地图整块可点击，点击后跳转至 SLG 大地图（沿用现有 SLG 地图界面，无需新做），玩家可在 SLG 地图查看每个都城详情、行军等
  • 玩法简介：用图标 + 一句话说明 "国家抢城" 与 "个人拿分" 两条主线规则；右上角 i 按钮入口提示玩家查看完整规则
  • 不显示宝箱与积分卡片（尚未开战）
3.3 战斗期形态（20:00 – 22:00）

主体显示战场地图+实时积分卡片，玩家可一眼看到当前局势与自身得分：

  • 战场地图（实时）：展示 9 座都城的实时占领归属，正在交战的城池高亮闪烁；同样可点击进入 SLG 大地图
  • 国家积分卡（上）：卡片水平拆分为左右两区。左区（国家积分）右上角显示总分，主体列出三类都城（内城 / 外城 / 远城）与本国占领情况，已占领的都城名绿色加“✓”高亮，并列出该类合计分；右区（国家奖励）以宝箱图标预览当前积分对应的宝箱品质（如“蓝色宝箱”）。点击卡片任意区域打开国家积分排行弹窗
  • 个人积分卡（下）：同样左右两区。左区（个人积分）右上角总分，主体两行公式：“击杀 28,400 × 1.0 = 28,400” / “损兵 32,333 × 0.3 = 9,700”；右区（个人奖励）以大号数字预览当前积分对应的宝箱数量（如 ×10）。点击卡片打开个人积分排行弹窗
  • 本国诶后看到的“关内 / 关外 / 敌境”名单不同，参见 2.3 节说明
  • 不显示宝箱（尚未结算）
国家 / 个人积分卡的外观与位置在战斗期与领奖期保持一致（均为左积分明细 / 右奖励预览的水平拆分卡），仅数据差异：战斗期为实时预览值，领奖期为最终结算值。
3.4 领奖期形态（22:00 – 次日 04:00）

主体恢复大宝箱悬浮居中形态（参考皇室战争宝箱界面）：暗色国战调背景，宝箱带浮动动画与光晕环。

  • 国家积分卡 / 个人积分卡：位于宝箱正下方，与战斗期同位同样式（水平拆分：左积分明细 / 右奖励预览），显示最终结算结果；占城 / 击杀 / 损兵明细仍以分类行与公式行呈现
  • 玩家无需手动充能：点击宝箱即可一键完成结算，自动按"先国家品质灌注、后个人数量加成"的顺序播放完整结算动画（光效 + 数字滚动 + 品质染色）
  • 结算动画播放完毕后，宝箱呈现最终品质与数量，并进入可领取状态
  • 两侧奖励卡片在结算期仍可点击，用于查看本场国家 / 个人最终得分与档位明细
领奖期内若玩家未点击宝箱，系统在次日 04:00 自动结算并通过邮件补发宝箱。
3.5 排行 / 奖励弹窗（战斗期 + 领奖期共用）

点击国家 / 个人积分卡片后弹出对应面板（参考 King Shot 得分奖励面板），弹窗结构如下：

  • 顶部金色标题栏 + 关闭按钮
  • 双页签：得分排行 和 奖励区间
  • 排行榜中，顶3名金银铜高亮；我方/我的条目用蓝色高亮

  • 弹窗底部固定吸附当前玩家积分与排名（参考 King Shot）
  • 点击遮罩层或 × 关闭弹窗
战斗期与领奖期共用同一套弹窗，仅数据快照不同。
3.6 领取宝箱

二宝箱界面 · 仓库与开启
2.1 宝箱主界面（v3.1 · 左上角等级 + 顶部进度条）
宝箱界面的入口位置（在游戏主界面的位置、按钮形态等）由游戏主界面统一承载，详见主界面相关文档；仓库入口同时存在于封地主界面与 SLG 主界面。本文档只描述进入宝箱界面后的内部结构与交互。

界面结构参考《兽化三国》与《咸鱼之王》宝箱页，自上而下划分为四层：标题栏（左上角宝箱等级徽章）、顶部进度条 + 兑换宝箱按钮、中央舞台（当前选中宝箱大图 + 切换箭头 + 开启按钮）、底部 5 品质宝箱栏。整体不挂接到主界面底部页签，作为独立全屏页存在。

  • 左上角宝箱等级徽章：独立于标题栏左上角，显示当前 Lv。当经验满足升级时叠加脉动红点；点击徽章进入宝箱等级查看界面（只列出已开放等级，剩余未开放以一条倒计时提示；列表上方附“等级影响积分奖励宝箱的品质”说明）。
  • 顶部进度条 + 兑换宝箱按钮：进度条上叠加 当前值 / 需求值 文本提示；右侧按钮未就绪时背景置灰（宝箱图仍正常上色），点击弹出“积分不足，无法兑换宝箱奖励”提示；就绪时亮起（黄绿色 + 黄色脉动光环）。循环机制不外露给玩家。
  • 中央舞台：带光晕 + 切换箭头，可循环切换 5 种品质，宝箱品质对应专属配色；库存为 0 时显示“暂无库存”占位，按钮自动禁用。不在舞台上展示概率提示文本，详细概率请点右上“?”按钮查看。
  • 中央开启按钮：单一按钮，实际开启数量 = min(当前品质库存, 15)，按钮文本动态为 “开启 X 个”（X 为实际会开的个数）。不允许一次超过 15 个，避免开箱动效过长。
  • 底部 5 槽：各品质库存数量；当前选中加金边高亮，库存>0 显示红点提示。
  • 返回按钮：点击后返回游戏主界面。
2.2 积分兑换宝箱 + 宝箱等级（两条独立赋能线）
⚠ 本节中所有带橙色虚线下划线的数值均为演示用占位值，仅用于帮助理解规则；最终数值以数值策划配置表为准。

玩家在宝箱仓库内开启宝箱时，同时贡献两条独立的赋能线：

  • 积分兑换：宝箱累积 积分 填充顶部进度条，填满后保持满状态不自动产出奖励；玩家点击兑换按钮后一次性消费所有可消费积分并产出多个奖励宝箱
  • 宝箱等级：宝箱累积 经验 提升 Lv，等级决定兑换产出的奖励宝箱品质；等级上限随赛季开放
一、积分兑换奖励宝箱（顶部进度条）
  • 积分获取：每开启 1 个宝箱按品质获得积分。演示值： 木制 +1 / 青铜 +10 / 白银 +20 / 黄金 +30 / 钻石 +50
  • 五段阈值循环（数值侧概念，不外露给玩家）：兑换时按顺序消费五段阈值，每消费一段产出 1 个奖励宝箱；五段走完后回到第 1 段循环。演示值： 10 → 20 → 30 → 40 → 60
  • 积分注入规则：积分注入进度条时，即使超过当前段阈值也不会自动扣除或进入下一段；进度条保持满状态并显示当前值/需求值（例：100/10）。仅当玩家点击兑换宝箱按钮时，一次性消费所有可消费的积分并产出对应数量的奖励宝箱（例：100 积分从第 1 段起 → 消费 10+20+30+40 = 100，产出 4 个奖励宝箱，进度重置为第 5 段 0/60）
  • 兑换动效：点击后多个奖励宝箱逐个从按钮位置飞到底部对应品质槽位，到达后该槽位数量 +1，同时顶部进度条逐步从“满”回退到剩余状态。不再弹出奖励庆祝弹窗，也不需要“收下”按钮。
  • 未就绪点击：按钮置灰状态下点击兑换按钮弹出 toast：“积分不足，无法兑换宝箱奖励”
  • 奖励宝箱品质：兑换产出的宝箱品质完全由当前宝箱等级决定（详见下表），与玩家实际开启的宝箱品质无关
二、宝箱等级（左上角徽章）
  • 经验获取：每开启 1 个宝箱同时获得经验，品质越高经验越多。演示值： 木制 +1 XP / 青铜 +5 XP / 白银 +10 XP / 黄金 +20 XP / 钻石 +40 XP
  • 升级触发：经验达到下一级需求后，左上角徽章出现脉动红点；点击徽章进入宝箱等级提升界面，点击升级按钮才正式提升一级（手动确认，便于玩家看到升级过程）

三、赛季上限与重置
  • 赛季上限：赛季初即开放全部 Lv 9 等级（兑换品质上限 钻石），不再划分上 / 下半期、无解锁倒计时
  • 新赛季重置：赛季切换时清空仓库中所有宝箱、重置宝箱等级为 Lv 1、重置积分与经验，仅结算奖励以邮件形式发放（如未领取）
  • 达到赛季上限：经验不再提升，升级按钮变为已满级；积分兑换仍按当前等级品质正常产出奖励宝箱
  • 未领取赛季宝箱：赛季结束时仓库中未开启的宝箱会自动开启，奖励以邮件形式发放（概率页脚注同步展示赛季倒计时）
2.3 开启流程与动效

点击开启 X 个（X = min(库存, 15)）后，进入全屏开箱层：暗场背景 + 聚光锥 + 居中宝箱开启动画 + 数量角标，随后切换到恭喜获得结果层（“奖励”金牌 + 卡片网格）。

  • 开启动画：渐显 → 聚光锥下降 → 宝箱微缩放 + 盖子上掀 + 火花飘出（约 1s）
  • 恭喜界面：奖励卡片按品质（普通 / 稀有 / 史诗 / 传说）配色，依次弹跳呈现
  • 底部按钮：再开 X 个（同品质，库存 > 0 才显示；X = min(剩余库存, 15)），点击空白处关闭恭喜界面
  • 若开启过程中触发了升级 / 满级奖励，收下后会自动弹出升级结算层，玩家"收下"以入库
